# LeadFlow CRM

LeadFlow CRM is a modern Client Lead Management System developed to help businesses organize and track customer leads efficiently. This project provides a simple and attractive dashboard interface for managing customer interactions and lead status.

## Features

- Secure Admin Login
- Dashboard Overview
- Lead Management System
- Search Functionality
- Lead Status Updates
- Responsive User Interface
- Client Tracking Dashboard

## Lead Status Types

- New
- Contacted
- Converted
